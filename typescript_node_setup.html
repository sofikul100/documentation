NODE JS SETUP WITH TYPESCRIPT

Install and Verify Node.js in Your Operating System

Go to nodejs.org and download Node.js.

Step 1: Create a Project Folder

mkdir nodejs-with-ts
cd nodejs-with-ts/

Step 2: Initialize a Node.js Project

npm init -y

Create a src folder in the root and in it create index.ts file with the following content:

console.log(2 + 3);

Step 3: Install TypeScript with Types

npm i -D typescript @types/node

Verify the installation by running:

npx tsc -v

Step 4: Create TypeScript Config File

npx tsc --init

Modify the generated tsconfig.json file to this:

{
  "compilerOptions": {
    "module": "CommonJS",
    "target": "ES2016",
    "noImplicitAny": true,
    "removeComments": true,
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "rootDir": "./src",
    "outDir": "./dist",
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}

Step 5: Install Other Packages We Use in the Project

Install express:

npm install express @types/express

Install dotenv and cors:

npm install cors dotenv

Install bcrypt and its types:

npm install bcrypt @types/bcrypt

Install prisma and @prisma/client:

npm install prisma @prisma/client

Then Write This Command npx prisma init

npx prisma init

Then Create a Model

model User {
    id        Int      @id @default(autoincrement())
    name      String
    email     String   @unique
    createdAt DateTime @default(now())
    updatedAt DateTime @updatedAt
}

Then just migrate it this command

npx prisma migrate dev --name add-updatedAt-to-user

Install build tools:

npm install --save-dev tsc-alias ts-node-dev tsconfig-paths

Step 6: Modify the Script in Your package.json File

{
  "scripts": {
    "dev": "tsnd --respawn -r tsconfig-paths/register --pretty --transpile-only ./src/index.ts",
    "build": "tsc && tsc-alias",
    "start": "node ./dist",
    "postinstall": "prisma generate"
  }
}

Step 7: Project File Structure

Define your project file structure as needed for your application.

Step 8: Create an Express Server

Modify your src/index.ts to this:

import express from "express";
require("dotenv").config();
const cors = require("cors");
const app = express();

app.use(cors());

const PORT = process.env.PORT || 8000;

app.use(express.json());

app.listen(PORT, () => {
  console.log(`Server is running on http://localhost:${PORT}`);
});

Summary of What the Code Does:

  • Express Setup: Sets up an Express server.
  • Environment Variables: Uses dotenv to manage sensitive configuration.
  • CORS Middleware: Allows cross-origin requests.
  • Port Configuration: Sets the server port from environment variables or defaults to 8000.
  • JSON Parsing: Parses incoming JSON payloads.
  • Server Start: Starts the server and logs the URL.

Step 9: Create a Simple API Endpoint

Add the following code to create an API endpoint in src/index.ts:

import express, { Request, Response } from "express";
app.get("/customers", (req: Request, res: Response) => {
  const customers = [
    { name: "John Doe", email: "john.doe@example.com", phone: "+1234567890" },
    { name: "Joel Smith", email: "joel.smith@example.com", phone: "+0987654321" },
  ];
 
  return res.status(200).json(customers);
});
